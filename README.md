# Zakat Calculator

A beautifully designed, Islamic-inspired web application built with React that helps Muslims calculate their annual Zakat obligation during Ramadan.

## Features
- Multi-currency support (USD, EUR, GBP, SAR, MYR, PKR, BDT, IDR)
- Comprehensive wealth calculation (assets, income & deductions)
- Live Nisab threshold (based on 85g of gold)
- Visual progress indicator
- Animated results and breakdowns
- Fully responsive design (works great on mobile)

## How It Works
The calculator follows the standard **Hanafi** methodology for Zakat — **2.5%** of wealth above the Nisab threshold.
